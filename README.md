# Metal Building 3D Configurator

A web-based 3D configurator for designing custom metal buildings with lean-tos, wrap-around corners, and various accessories.

## Features

- **Interactive 3D View**: Rotate, zoom, and pan to view your building from any angle
- **Customizable Dimensions**: Set width, length, eave height, and roof pitch
- **Lean-To Support**: Add lean-tos on any side (left, right, front, back)
- **Wrap-Around Corners**: Connect adjacent lean-tos with seamless corner geometry
- **Roof Overhangs**: Configure eave and gable overhangs
- **Color Customization**: Choose colors for walls, roof, and trim
- **Accessories**: Add walk doors, roll-up doors, windows, and frame-outs
- **Real-time Updates**: See changes instantly as you configure

## Usage

Simply open `index.html` in a modern web browser. No server or build process required.

### Controls
- **Drag**: Rotate the view
- **Scroll/Pinch**: Zoom in/out
- **Tap accessories**: Select and reposition

## Technology

Built with:
- Three.js for 3D rendering
- Pure HTML/CSS/JavaScript (no build tools required)

## Browser Support

Works in all modern browsers that support WebGL:
- Chrome
- Firefox
- Safari
- Edge
